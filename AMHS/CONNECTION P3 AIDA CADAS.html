C. Points de correspondance critiques

Points critiques (doivent correspondre exactement) :
1. Valeurs PSEL/SSEL (0xff dans AIDA = 'f'H dans CADAS)
2. Valeurs TSAP (5033 dans AIDA = '5033'H dans CADAS)
3. Nom/Mot de passe MTA provenant des paramètres globaux AIDA
4. Mot de passe utilisateur MTS de LA AIDA vers la boîte aux lettres CADAS
5. Les adresses IP doivent être correctes et opérationnelles
Conversion Hexadécimal vers Décimal :
• 'f'H (hex) = 15 (décimal) = 0x0f (en notation AIDA)
• '5033'H (hex) = 20531 (décimal) mais CADAS l'utilise comme chaîne, AIDA utilise 5033
• Vérifier les valeurs réelles dans la documentation des deux systèmes
