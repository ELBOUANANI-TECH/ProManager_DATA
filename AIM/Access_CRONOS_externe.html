Introduction

Cette procédure décrit les étapes pour accéder au système CRONOS depuis un réseau externe via le pare-feu Barracuda.

⚠️ Attention : Cet accès est réservé au personnel autorisé. Assurez-vous de disposer des credentials valides avant de commencer.
1. Connexion au Firewall Barracuda
1

Ouvrez votre navigateur web et accédez à l'adresse du pare-feu Barracuda :

https://172.21.0.251
Cette adresse est l'interface de gestion du pare-feu Barracuda. Une connexion sécurisée (HTTPS) est requise.
2

Entrez vos identifiants d'authentification :

Capture d'écran : Interface de connexion Barracuda
🛡️
Page de connexion Barracuda Firewall
URL: https://172.21.0.251
2. Configuration de la route statique

Après connexion au pare-feu Barracuda, accédez à la section de configuration réseau :

BARC
ADC01 - Production
TRAFFIC
SECURITY
ACCESS CONTROL
NETWORK
ADVANCED

Add Static Route

IPv4
IP Address 192.168.112.0
Netmask 255.255.255.0
Gateway Address 192.168.12.201
Network Interface 98-1-6
IP ADRESS : Réseaux Local de l'aéroport Destination 192.168.1XX.0
Note importante : Cette configuration route le trafic vers le réseau local de l'aéroport de destination (192.168.1XX.0).
3. Accès à CRONOS
1

Via une position CADAS de l'aéroport de destination

Position CADAS :
Il s'agit d'un poste de travail local dans l'aéroport de destination, pré-configuré pour accéder au réseau interne.
2

Accédez à l'interface CRONOS :

https://172.21.31.125/cronos-portal
Cette adresse est l'interface CRONOS accessible depuis le réseau interne après configuration de la route.
3

Entrez vos identifiants CRONOS :

Capture d'écran : Interface CRONOS Portal
🌐
Page d'accueil CRONOS Portal
URL: https://172.21.31.125/cronos-portal
Résumé de la configuration
Élément Valeur Description
Firewall Barracuda 172.21.0.251 Point d'entrée externe
Réseau destination 192.168.112.0/24 Réseau local de l'aéroport
Passerelle 192.168.12.201 Gateway pour le réseau destination
Interface 98-1-6 Interface réseau sur le firewall
CRONOS Portal 172.21.31.125 Adresse interne de CRONOS
✅ Connecté avec succès
