Ajouter un utilisateur Aerodb au domaine AIM

Domaine : ondaids.local

1. Enlever la position de l'ancien domaine

1
Appuyez sur Windows + r, tapez sysdm.cpl et appuyez sur Entrée.
2
Cliquez sur le bouton Modifier dans l'onglet « Nom de l'ordinateur ».
3
Sélectionnez Groupe de travail et entrez WORKGROUP.
Computer Name/Domain Changes
Workgroup:
WORKGROUP
4
Cliquez sur OK, confirmez et redémarrez l'ordinateur.
Il vous sera demandé de redémarrer l'ordinateur afin que les modifications puissent être appliquées.

2. Modifier l'adresse DNS

Chaque position est connectée via deux cartes réseau. Pour éviter un conflit d'adresse, allouez à la deuxième carte une adresse IP via DHCP.

Étape 1 : Accéder aux paramètres réseau

1
Menu Démarrer > Paramètres > Réseau et Internet.
2
Cliquez sur Modifier les options de l'adaptateur (Paramètres réseau avancés).

Étape 2 : Configurer les propriétés réseau

3
Cliquez droit sur votre carte active > Propriétés.
4
Sélectionnez Internet Protocol Version 4 (TCP/IPv4) > Propriétés.
5
Cliquez sur Utiliser les adresses de serveur DNS suivantes et entrez :
DNS préféré :
172.21.31.180
DNS alternatif :
172.21.31.180
6
Redémarrer l'ordinateur.

3. Ajouter la position au nouveau domaine

1
Suivre les mêmes étapes que la première partie jusqu'au point où on doit entrer le nom du domaine ondaids.local.
Computer Name/Domain Changes
Domain:
ondaids.local
2
Entrez les informations d'identification du domaine :
Nom d'utilisateur :
idsuser
Mot de passe :
123Ondapass
3
Cliquez droit sur Démarrer > Computer management.
4
Accédez à Utilisateurs et groupes locaux et ajoutez l'utilisateur Ondaids\lamuser au groupe Administrateurs.
5
Vérifiez les informations d'identification du domaine : idsuser/123Ondapass.
6
Redémarrer l'ordinateur.
7
Se connecter avec le nouvel utilisateur.
8
Connecter l'utilisateur de la position à la base de données.

4. Connexion à la base de données

a
Ouvrir la console cmd et taper :
cd \ids\DAF\Startup\
AddAdministrator.bat USER_ID USER_ID TNS_SERVICE_NAME
Les user ID et TNS Service Name viennent du chemin suivant :
Menu démarrer > ODBC Data Sources > Sources de données système > ONDA
b
Explorateur de fichiers > Cliquez droit sur Ce PC > Connecter un lecteur réseau :
  • Lecteur : O:
  • Dossier : \\172.21.31.67\license
  • Identifiants : idsuser/1230ndapass
